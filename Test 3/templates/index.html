Facebook Page Insights Tool

Chào mừng bạn đến với công cụ phân tích Facebook Page Insights. Vui lòng nhập Access Token của bạn để bắt đầu.

Lưu ý quan trọng:

  • Access Token của bạn cần có các quyền sau để hoạt động đầy đủ: pages_show_list (để liệt kê trang) và pages_read_engagement (để xem insights).
  • Bạn có thể lấy Access Token từ Facebook Developer Tools.
